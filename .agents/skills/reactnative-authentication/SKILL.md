# Passos
1. Identifique os requisitos de funcionalidade móvel (navegação, módulos nativos, objetivo de performance) e escolha as bibliotecas adequadas.
2. Escreva ou modifique componentes usando os primitivos do React Native e padrões recomendados (ex: FlatList, useState).
3. Integre módulos específicos da plataforma (câmera, geolocalização) via Expo ou módulos nativos, tratando devidamente as permissões.
4. Teste o aplicativo em simuladores/dispositivos iOS e Android e ajuste as diferenças entre as plataformas e a performance.
5. Prepare e documente as etapas de implementação (pipeline de CI/CD, diretrizes de submissão da loja de apps) e acompanhe o feedback do usuário.

# Limites
- Não implemente lógicas no servidor nem use componentes restritos para a web; mantenha o foco no código multiplataforma para dispositivos móveis.
